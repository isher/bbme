轻量级虚拟机部署管理工具-Multipass

Multipass是一种简单的虚拟机工具。它不仅使启用虚拟机变得快速简易，还使管理那些虚拟机变得异常简单，因此可以立即开始针对云、边缘、物联网或任何一种类型的技术进行开发。实际上，Multipass包含一个系统任务栏工具，你只要点击一下就可以启动和停止虚拟机，甚至进入虚拟机的外壳。支持Linux、Windows系统等。

一、Multipass介绍

Multipass是一种简单的虚拟机工具。它不仅使启用虚拟机变得快速简易，还使管理那些虚拟机变得异常简单，因此可以立即开始针对云、边缘、物联网或任何一种类型的技术进行开发。实际上，Multipass包含一个系统任务栏工具，你只要点击一下就可以启动和停止虚拟机，甚至进入虚拟机的外壳。支持Linux、Windows系统等。

二、Multipass环境搭建

1.Ubuntu系统搭建

  1. sudo apt update
  2. sudo apt install snapd           #安装snapd
  3. sudo snap install multipass      #安装multipass

2.Centos系统搭建

  1. sudo yum install epel-release
  2. sudo yum install snapd
  3. sudo systemctl enable --now snapd.socket
  4. sudo ln -s /var/lib/snapd/snap /snap
  5. sudo snap install multipass

三、Multipass常用命令

1.查找镜像

  1. multipass find

2.创建虚拟机

语法：multipass launch -n 虚拟机名称

-n, --name: 名称
-c, --cpus: cpu核心数, 默认: 1
-m, --mem: 内存大小, 默认: 1G
-d, --disk: 硬盘大小, 默认: 5G

  1. multipass launch -n ubuntu-lts -c 4 -m 4G -d 40G

3.进入虚拟机

语法：multipass shell 虚拟机名称

  1. multipass shell ubuntu-lts

4.直接使用虚拟机

语法：multipass exec 虚拟机名称  --命令

  1. multipass exec ubuntu-lts -- ls

5.查看虚拟机列表

  1. multipass ls
  2. multipass list

6.查看虚拟机信息

语法：multipass info 虚拟机名称

  1. multipass info ubuntu-lts

7.重启虚拟机

语法：multipass restart 虚拟机名称

  1. multipass restart ubuntu-lts

8.删除虚拟机

语法：multipass delete 虚拟机名称

--purge  彻底删除

  1. multipass delete ubuntu-lts
  2. multipass delete --purge ubuntu-lts 彻底删除

9.恢复删除虚拟机

语法：multipass recover 虚拟机名称

  1. multipass recover ubuntu-lts

10.启动虚拟机

语法：multipass start 虚拟机名称

  1. multipass start ubuntu-lts

11.暂停虚拟机

语法：multipass stop 虚拟机名称

  1. multipass stop ubuntu-lts

12.宿主机挂载虚拟机

语法：multipass mount 宿主机目录 虚拟机名称：虚拟机目录

  1. multipass mount /mnt ubuntu-python
  1. multipass mount /mnt ubuntu-python:/mnt

13.宿主机卸载虚拟机

语法：multipass unmount 虚拟机名称

  1. multipass unmount ubuntu-lts

14.挂起虚拟机

语法：multipass suspend 虚拟机名称

  1. multipass suspend ubuntu-lts

15.获取版本信息

  1. multipass version

16.帮助

  1. multipass help
