Windows 11 专业版 

YVNC2-3HGX3-YQBQR-T89DQ-WQK2K (自动激活密钥或更换激活密钥)

W269N-WFGWX-YVC9B-4J6C9-T83GX （GVLK 默认安装密钥，win10也用这个）

========以下为win10激活控制台命令=========

安装win10 

//命令中的第一个@改为win10镜像虚拟光驱的盘符，第二个@改为U盘盘符。 //运行命令，等待大概10来分钟即可（取决于U盘写入速度）。

 dism /Split-Image /ImageFile:@:sourcesinstall.wim /SWMFile:@:sourcesinstall.swm /FileSize:3800

安装后的激活，采用qsis.cn的KMS服务器，打开管理员CMD，运行如下命令

  1. slmgr -upk 卸载秘钥
  2. slmgr -ipk 安装kms秘钥
  3. slmgr /skms 设置KMS
  4. slmgr /ato 激活

slmgr /ckms
slmgr /ipk W269N-WFGWX-YVC9B-4J6C9-T83GX
slmgr /spri
slmgr /act-type 2
slmgr /skms kms.qsis.cn
slmgr /ato
slmgr /dlv

如果你之前用过此方法，并且已经使用了一段时间，再次出现已经到期的情况，请按照下列步骤执行


鼠标右键点击“开始”，选择windows powershell（管理员）

现在分两种情况执行两种不同方式激活命令

1、之前使用我这里的批量程序或上述代码激活的，可以直接输入或者直接复制slmgr /ato命令到powershell管理员窗口，回车即可

2、如果不是使用以上命令激活的，需要依次复制下面命令

slmgr /act-type 2
slmgr /skms kms.qsis.cn
slmgr /ato
slmgr /dlv
最后看到“成功地激活了产品”，就可以关闭相应窗口，继续你的工作了。

与windows激活不同，office的vol必须进入安装目录，因为要调用ospp，所以目录在：C:/Program Files (x86)/Microsoft Office/Office16下

建立批处理文件：内容如下

cscript ospp.vbs /sethst:kms.qsis.cn
cscript ospp.vbs /act
cscript ospp.vbs /dstatus

保存，打开office，输入key后关闭，然后运行批处理文件


==========Office 更换key序列号 的方法===================
cscript ospp.vbs /dstatus 先查看当前安装的密钥的最后五位是什么，将看到诸多内容中的如下信息

Last 5 characters of installed product key: XXXXX

卸载密钥

cscript ospp.vbs /unpkey:xxxxx （xxxxx为秘钥最后五位，查询 可以使用 cscript ospp.vbs /dstatus命令）

插入新的密钥

cscript ospp.vbs /inpkey:XQNVK-8JYDB-WJ9W3-YJ8YR-WFG99 此处以office2016 Vol版本为例

cscript ospp.vbs /dstatus 重新查看状态，如果ok，将看到

        DNS auto-discovery: KMS name not available
        KMS machine registry override defined: kms.qsis.cn:1688
        Activation Interval: 120 minutes
        Renewal Interval: 10080 minutes
        KMS host caching: Enabled


========如果你装的是零售版本或者其他版本，那么你需要变更为VOL版本，方法如下=========

将以下内容建立批处理文件，以管理员权限运行

@echo off
:ADMIN
openfiles >nul 2>nul ||(
echo Set UAC = CreateObject^("Shell.Application"^) > "%temp%getadmin.vbs"
echo UAC.ShellExecute "%~s0", "", "", "runas", 1 >> "%temp%getadmin.vbs"
"%temp%getadmin.vbs" >nul 2>&1
goto:eof
)
del /f /q "%temp%getadmin.vbs" >nul 2>nul
for /f "tokens=6 delims=[]. " %%G in ('ver') do set win=%%G
pushd "%~dp0"
Title Office 2016 Retail to Volume License Converter
echo Press Enter to start VL-Conversion...
echo.
pause
echo.
cd /D "%SystemRoot%System32"
if %win% GEQ 9200 (
cscript slmgr.vbs /ilc "%ProgramFiles%Microsoft OfficerootLicenses16ProPlusVL_KMS_Client-ppd.xrm-ms"

============OFFICE2019 脚本激活======================

@echo off
(cd /d "%~dp0")&&(NET FILE||(powershell start-process -FilePath '%0' -verb runas)&&(exit /B)) >NUL 2>&1
title Office 2019 Activator r/Piracy
echo Converting... & mode 40,25
(if exist "%ProgramFiles%Microsoft OfficeOffice16ospp.vbs" cd /d "%ProgramFiles%Microsoft OfficeOffice16")&(if exist "%ProgramFiles(x86)%Microsoft OfficeOffice16ospp.vbs" cd /d "%ProgramFiles(x86)%Microsoft OfficeOffice16")&(for /f %%x in ('dir /b ..rootLicenses16ProPlus2019VL*.xrm-ms') do cscript ospp.vbs /inslic:"..rootLicenses16%%x" >nul)&(for /f %%x in ('dir /b ..rootLicenses16ProPlus2019VL*.xrm-ms') do cscript ospp.vbs /inslic:"..rootLicenses16%%x" >nul)
cscript //nologo ospp.vbs /unpkey:6MWKP >nul&cscript //nologo ospp.vbs /inpkey:NMMKJ-6RK4F-KMJVX-8D9MJ-6MWKP >nul&set i=1
:server
if %i%==1 set KMS_Sev=kms.qsis.cn
cscript //nologo ospp.vbs /sethst:%KMS_Sev% >nul
echo %KMS_Sev% & echo Activating...
cscript //nologo ospp.vbs /act | find /i "successful" && (echo Complete) || (echo Trying another KMS Server & set /a i+=1 & goto server)
pause >nul
exit
