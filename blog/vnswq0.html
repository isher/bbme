创维E900V21E海思3798mv310WiFi-mt7668rsn不刷机破解

总之这是几大运营商的产品，从2018年开始就已经向市场上投放，其中型号没变，但配置已经变更了很多次，而且各个厂商的都有，芯片混用严重。

今天说的这款是北京电信的E900V21E海思3798mv310WiFi-mt7668rsn(无读卡器插槽版本) 右上角写的4kFull,后面是三个螺丝的，侧面给2个usb口，和一个sd卡插口，有的版本没有SD的卡插口（外观也都差不多，就不多做介绍了），软破解问题不大，不会涉及到硬件故障，2021年下半年上市的版本。

创维E900V21E这款设备虽然是一款阉割了SD卡槽的设备，不过总体配置上来看还是可以的。很多人都在找这款机器的刷机包，而实际上这款机器根本就不需要刷机包！！！

北京电信的盒子本地设置密码为10000，此盒子有一个严重bug，即在设置->高级设置->显示设置中，一定要把最佳显示模式的默认勾选关闭，否则无法上到4K30HZ。

创维E900V21E开ADB的方法：系统设置->其他设置->有测试随便一个选项不用管，按5下遥控器邮件，会突然多出一个【usb调试】选项，打钩就行了，这时候，你的adb connect ip 就能链接通了。（同样在这个设置->其他设置，点击左键5下，出现 远程控制管理。），同时记得链接wifi，并记下wifi的ip地址。

PC端，打开cmd（希望你已经拷贝过adb的几个文件了）

命令：

adb connect IP(你的盒子ip)，#链接到盒子
adb shell #登录到盒子
isher@Hi3798MV310:/ $su root #  如果不是root权限，执行这个
root@Hi3798MV310:/ # mount -o rw,remount /system  #默认情况下system不可写，重新挂载可写权限。


将当贝桌面apk拷贝到存储卡，例如db.apk.并将存储卡插入

root@Hi3798MV310:/ # cd /mnt/sda/sda1/  #cd到存储卡或

者移动硬盘的挂载位置

root@Hi3798MV310:/ #cd /mnt/sda/sda1/

root@Hi3798MV310:/mnt/sda/sda1# cp db.apk /system/app/   #拷贝当贝桌面到系统目录
root@Hi3798MV310:/mnt/sda/sda1# cp es.apk /system/app/ #拷贝Es管理器到系统目录

-------------下面内容可以不执行，执行上述内容重启后即可-------------------
root@Hi3798MV310:/ #pm install db.apk  #安装当贝桌面
root@Hi3798MV310:/ #am start com.dangbei.tvlauncher  #启动当贝桌面
